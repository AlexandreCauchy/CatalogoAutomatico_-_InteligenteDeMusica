# SonicHub - Catálogo Inteligente de Música

Bem-vindo ao SonicHub, um sistema de catalogação musical inteligente desenvolvido com tecnologias web puras.

## Objetivo
O projeto visa fornecer uma interface premium e funcionalidades de IA (análise de timbre) sem depender de bibliotecas externas, permitindo total controle e performance.

## Funcionalidades
- **Catalogação Automática**: Organização de músicas por metadados.
- **Análise de Timbre Vocal**: Reconhecimento de autoria via assinatura sonora.
- **Interface Premium**: Design responsivo com efeitos de glassmorphism.

## Estrutura de Arquivos
- `index.html`: Estrutura principal.
- `css/estilo.css`: Estilização visual.
- `js/catalogo.js`: Lógica de dados e análise.
- `js/aplicativo.js`: Interface e interações.
- `documentacao/`: Pasta com manuais detalhados.
