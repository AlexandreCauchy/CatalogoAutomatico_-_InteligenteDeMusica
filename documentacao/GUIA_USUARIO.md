# Guia do Usuário - SonicHub

Como utilizar o seu catálogo inteligente de música.

## 1. Adicionar Músicas
Clique no ícone de "+" no topo para adicionar uma música manualmente ou via análise.

## 2. Análise Vocal
Ao carregar uma música desconhecida, o SonicHub analisará o timbre automaticamente. Se ele encontrar um autor compatível no seu catálogo, uma sugestão de movimentação aparecerá.

## 3. Pesquisa e Filtros
Utilize a barra de pesquisa no topo para encontrar rapidamente artistas, álbuns ou títulos.

## 4. Organização
O sistema mantém "pastas" virtuais baseadas nos nomes dos autores.
